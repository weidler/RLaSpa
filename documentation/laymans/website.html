Problem statement and motivation:

Reinforcement Learning (RL) is a subfield of the artificial intelligence (AI) where an agent is set free in an unknown environment, where will receive positive or negative rewards depending on the actions it makes. In this situation, the agent will act using trial and error with the objective of obtain the maximum reward possible and learn the way to do so consistently. This learning obtained from the interaction, from the experience is called policy.
